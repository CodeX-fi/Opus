ReplaceNumberPlaceholderWithIndex<T>(String, T, Nullable<IEnumerable<T>>, Nullable<ILogbook>)

Replace the number placeholder of a string with the index value of a given item.

If needed and if an enumerable of all items is provided, additional zeroes will be added to even out the number of numbers (01, 02 .. 001, 002.. etc).
